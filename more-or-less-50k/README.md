# Приложение, определяющее превысит ли средний заработок человека порог $50k

Вы вводите параметры в приложении, приложение вам на основе ваших введенных данных отвечает, превысит ваш средний заработок $50k или нет

Запускается командой

```
streamlit run src/app.py 
```

Задеплоенное приложение - https://more-or-less-50k.streamlit.app/
